☁️ 깃허브 실시간 연동 등록 방법 (무료)

                
            

            

                
8명이 카톡으로 접속하여 실시간 동기화하려면 구글 Firebase 가입 후 설정값 입력이 필요합니다.

                
                
    
                    
  1. 구글에서 Firebase Console에 접속한 뒤 구글 계정으로 로그인합니다.
    2. 
                    
  2. [프로젝트 추가]를 선택하고 원하는 이름(예: jajang-order)을 적어 프로젝트를 생성합니다.
    3. 
                    
  3. 생성 후 대시보드 화면 중앙의 </> (웹앱) 아이콘을 클릭하고 별칭을 입력해 앱을 등록합니다.
    4. 
                    
  4. 대시보드가 완료되면 아래 형식의 firebaseConfig 키 값들이 화면에 나옵니다.
    5. 
                    
  5. 해당 내용들을 복사한 다음, 이 파일(index.html) 소스코드 304번째 줄에 있는 MY_FIREBASE_CONFIG 변수 내부에 동일하게 붙여넣고 깃허브에 저장해 주세요.
    6. 
                    
  6. [중요 설정] Firebase 왼쪽 메뉴에서 Firestore Database를 클릭해 [데이터베이스 생성]을 진행해 주세요. (위치 기본값 지정, 보안 규칙은 '테스트 모드에서 시작'을 권장합니다)
